Alpha Phi Omega Rho Pi

This is the website for Alpha Phi Omega, Rho Pi Chapter, a community service based professional fraternity at UCSD. It is a Ruby on Rails web application used by over 120 active members for keeping track of member requirements, signing up for events, and tracking attendance. My contributions include implementing a mobile calendar view, adding the option to mark members late to events and give them half the possible attendance credit, and marking the total number of drivers signed up for events that go off campus.
